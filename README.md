# Simple Link Opener (GUI App)

A simple desktop application built with Python and Tkinter that allows users to open any web link directly from the app.

## Features

- Open any website link in the browser
- Simple and user-friendly GUI
- Input validation (checks if field is empty)
- Success and error messages
- Built using Tkinter

## How to Run

```bash
python link_opener.py
```

> Make sure the file name matches your Python file name.

## How it works

1. Enter a website link (e.g. google.com)
2. Click "Open Link"
3. The link will open in your default browser
4. A message will show if the action was successful or if input is empty

## Technologies Used

- Python
- Tkinter (GUI)
- Webbrowser module

## Example

```
Input: google.com
Output: Opens Google in browser
```
